Étape(s) d'installation
  • Double-cliquez sur le fichier
  • Cliquez sur "installer"
  • Allez dans le menu de démarrage, section "Jeux", sous-section "Jeux de tir"
  • Sélectionnez "Truly Unusual Experience of Revolution"
  • Décompressez le fichier
  • Allez dans le répertoire "tuer"
  • Double-cliquez sur "tuer"
  • Allez dans les Préférences Système
  • Sélectionnez Sécurité & Confidentialité
  • Cliquez sur le verrou pour faire des changements et
    autoriser en tant qu'administrateur (si nécessaire)
  • Sélectionnez “Autoriser les applications téléchargées de:
    N'importe où”
  • Double-cliquez sur le fichier
  • Suivez les instructions de l'installeur
  • Allez dans le menu de démarrage dans la section précédemment choisie
  • Sélectionnez "tuer"
  • Décompressez le fichier
  • Allez dans le répertoire "tuer"
  • Double-cliquez sur "tuer.bat"
