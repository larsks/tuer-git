Installation step(s)
  • Double-click on the file
  • Click on "install"
  • Go to the start menu, section "Games", sub-section "Shooters"
  • Select "Truly Unusual Experience of Revolution"
  • Uncompress the file
  • Go into the directory "tuer"
  • Double-click on "tuer"
  • Go to the System Preferences
  • Select Security & Privacy
  • Click the lock to make changes and
    authorise as an admin (if necessary)
  • Select “Allow applications downloaded from:
    Anywhere”
  • Double-click on the file
  • Follow the instructions of the installer
  • Go to the start menu in the previously chosen section
  • Select "tuer"
  • Uncompress the file
  • Go into the directory "tuer"
  • Double-click on "tuer.bat"
